No software product is free of bugs and can therefore always be improved. This is especially true for the currently prototypical implementation of the Open eCard App.

Against this background, the Open eCard Team is gladly receiving bug reports and proposals and is working hard to address them in time and to provide feedback for the input.

The Open eCard Project is providing an issue tracker at http://openecard.org/issues in order to receive and manage the bug reports and proposals.
